## 8. Glossary

### 8.1 Common Terms

| Term                  | Definition                                                              |
| --------------------- | ----------------------------------------------------------------------- |
| **mAP**               | Mean Average Precision - metric untuk object detection accuracy         |
| **IoU**               | Intersection over Union - overlap antara predicted dan ground truth box |
| **NMS**               | Non-Maximum Suppression - menghilangkan duplicate detections            |
| **Anchor Box**        | Predefined box shapes untuk detection                                   |
| **Feature Map**       | Output dari convolutional layer                                         |
| **Backbone**          | Bagian CNN untuk feature extraction                                     |
| **Head**              | Bagian untuk prediction (classification + localization)                 |
| **FPN**               | Feature Pyramid Network - multi-scale detection                         |
| **Transfer Learning** | Menggunakan pretrained weights untuk task baru                          |
| **Fine-tuning**       | Training ulang sebagian model untuk task spesifik                       |

### 8.2 Training Terms

| Term               | Definition                                        |
| ------------------ | ------------------------------------------------- |
| **Epoch**          | Satu kali pass seluruh training data              |
| **Batch**          | Subset data yang diproses bersamaan               |
| **Learning Rate**  | Step size untuk gradient descent                  |
| **Loss**           | Nilai error yang diminimasi                       |
| **Gradient**       | Turunan loss terhadap parameters                  |
| **Optimizer**      | Algoritma untuk update weights (SGD, Adam, AdamW) |
| **Regularization** | Teknik mencegah overfitting                       |
| **Augmentation**   | Teknik memperbanyak data secara artificial        |
| **Early Stopping** | Stop training saat performance tidak improve      |
| **Checkpoint**     | Saved state dari model selama training            |

### 8.3 Detection-Specific Terms

| Term                 | Definition                                                |
| -------------------- | --------------------------------------------------------- |
| **Bounding Box**     | Kotak pembatas objek [x, y, width, height]                |
| **Confidence Score** | Probability bahwa deteksi benar                           |
| **Ground Truth**     | Label yang benar (human annotated)                        |
| **True Positive**    | Deteksi benar                                             |
| **False Positive**   | Deteksi salah (object tidak ada)                          |
| **False Negative**   | Object yang tidak terdeteksi                              |
| **Precision**        | TP / (TP + FP) - seberapa akurat positive predictions     |
| **Recall**           | TP / (TP + FN) - seberapa banyak positives yang ditemukan |
