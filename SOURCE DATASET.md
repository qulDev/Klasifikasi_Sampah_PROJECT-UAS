# Panduan Lengkap Dataset Klasifikasi Sampah Anorganik untuk Program Python Real-Time

Berikut ini adalah daftar lengkap **dataset sampah anorganik** yang dapat Anda unduh dan gunakan untuk melatih model klasifikasi sampah real-time menggunakan Python, disertai dengan link download langsung dan penjelasannya.

## Dataset Terbesar dan Paling Recommended

### 1. **Waste Classification v2** (27,500+ Gambar) ⭐ PALING RECOMMENDED

**Link Download:** https://www.kaggle.com/datasets/sapal6/waste-classification-data-v2

- **Jumlah Gambar:** ~27,500 citra terstruktur
- **Kategori:** 3 kelas (Organic/O, Recyclable/R, Nonrecyclable/N)
- **Format:** Sudah terpisah Train/Test
- **Split:** Train 87%, Valid 8%, Test 5%
- **License:** CC BY-SA 4.0
- **Kelebihan:** Dataset sudah terstruktur rapi, siap pakai untuk training langsung


### 2. **Waste Classification Data v1** (25,000+ Gambar)

**Link Download:** https://www.kaggle.com/datasets/techsash/waste-classification-data

- **Jumlah Gambar:** ~25,077 citra
- **Kategori:** 2 kelas (Organic, Recyclable)
- **Format:** Sudah terbagi Train/Test
- **Split:** Train 22,564, Test 2,513
- **License:** CC BY-SA 4.0


### 3. **Garbage Dataset (V2)** (19,762 Gambar)

**Link Download:** https://www.kaggle.com/datasets/sumn2u/garbage-classification-v2

- **Jumlah Gambar:** 19,762 citra
- **Kategori:** 10 kelas (Metal, Glass, Biological, Paper, Battery, Trash, Cardboard, Shoes, Clothes, Plastic)
- **Format:** Terstruktur dengan folder per kategori
- **Kelebihan:** Kategori lebih detail, cocok untuk deteksi multi-kelas


### 4. **RealWaste Dataset** (4,752 Gambar)

**Link Download:** https://archive.ics.uci.edu/dataset/908/realwaste

- **Jumlah Gambar:** 4,752 gambar resolusi 524x524
- **Kategori:** 9 kelas (Cardboard, Food Organics, Glass, Metal, Miscellaneous Trash, Paper, Plastic, Textile Trash, Vegetation)
- **Sumber:** UCI Machine Learning Repository
- **Kualitas:** Diambil dari lingkungan landfill nyata


### 5. **Waste Classification Dataset** (4,000 Gambar)

**Link Download:** https://www.kaggle.com/datasets/adithyachalla/waste-classification

- **Jumlah Gambar:** 4,000 citra
- **Kategori:** 9 kelas (Cardboard, Food Organics, Glass, Metal, Miscellaneous Trash, Paper, Plastic, Textile Trash, Vegetation)
- **Format:** Folder terstruktur per kategori
- **Cocok untuk:** Transfer learning dan rapid prototyping

## Cara Menggunakan Dataset untuk Program Python Real-Time

### Struktur Folder yang Direkomendasikan:```

├── datasets/
│   ├── train/
│   │   ├── organic/
│   │   ├── inorganic/

│   ├── test/
│   └── validation/

├── src/
│   ├── train.py
│   ├── detect.py
│   └── preprocess.py
└── requirements.txt

```

### Kode Python Contoh untuk Real-Time Classification:
```python
import cv2
from ultralytics import YOLO

# Load model
model = YOLO('best.pt')

# Real-time detection
cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    if ret:
        results = model(frame)
        
        # Render results
        annotated_frame = results[^0].plot()
        cv2.imshow('Waste Detection', annotated_frame)
        
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break

cap.release()
cv2.destroyAllWindows()
```## Ringkasan Rekomendasi

**Untuk pemula:**
- Gunakan **Waste Classification v2** atau **Garbage Dataset (V2)**
- Ukuran dataset besar, sudah terstruktur, mudah digunakan

**Untuk deteksi real-time dengan YOLO:**
- Gunakan **SISCER-Project** (Indonesian, 2-kelas)
- Format mendukung YOLOv8/v5 langsung

**Untuk kategori detail:**
- Gunakan **waste_pictures** (24 kategori)
- Atau **TrashBox Dataset** (7 kategori spesifik)

**Untuk performa terbaik:**
- Kombinasikan beberapa dataset
- Lakukan data augmentation
- Gunakan transfer learning dari model pre-trained

Semua dataset di atas dapat diunduh langsung dari link yang disediakan. Pastikan Anda memiliki kapasitas penyimpanan yang cukup sebelum mengunduh dataset besar!
